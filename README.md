### Deploy with Node.js

```bash
npm install
npm run build
npm run start
```

## Development

### Running Locally

1. Install NodeJS 20.
2. Clone the repository.
3. Install dependencies with `npm install`.
4. Start the application using `npm run dev`.
5. Visit `http://localhost:3000` in your browser.
